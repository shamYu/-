Orm 映射处理类

之前一直都是操作对象，orm就是通过实体类完成 CRUD 操作。

定义实体类

@Table(name="T_YDZF_FJXX")
public class FileInfo {
    @Id
    @Column(name="XH")
    private String id; //文件id
    @Column(name="FJMC")
    private String name;//文件名称
    @Column(name="FJDX")
    private String size; //文件大小
    @Column(name="ZJXH")
    private String relationId;//关联ID
    @Column(name="FJLX")
    private String type; //文件分类类型
    @Column(name="FJDZ")
    private String filePath; //文件路径
    @Column(name="FJGS")
    private String fileType; //文件格式
    @Column(name="SCSJ")
    private Date uploadTime;//上传时间
    @Column(name="CJR")
    private String uploader;//上传人
    @Column(name="FJMS")
    private String description; //文件描述
}

实体类 Table 定义映射表名， @Id定义主键，@Column 定义列名。

增删改查操作

public class FileService extends GenericService{
    private Orm<FileInfo> orm = new Orm<FileInfo>(FileInfo.class);

    public List<FileInfo> getFileList(String id, String type){
        return orm.bindWhere("relationId", id).bindWhere("type", type).search();
    }
}

orm 支持链式调用， 使用bindWhere 绑定参数， 最后可以执行 search , delete 操作。

保存数据

FileInfo fileInfo = new FileInfo();
//省略设置值
orm.save(fileInfo);

//事务处理
Connection con = dao.getConnection();
con.setAutoCommit(false);
orm.save(fileInfo, con);
//。。。。
con.commit();

删除数据

orm.delete("111") //根据主键删除

//根据条件删除
orm.bindWhere("relationId", id).bindWhere("type", type).delete();

//事务处理
Connection con = dao.getConnection();
con.setAutoCommit(false);
orm.delete(con);
orm.delete("111", con);
//。。。。
con.commit();

查询数据


FileInfo fileInfo = orm.get("111111"); //主键获取 FileInfo 对象

//根据条件查询
List<FileInfo> list = orm.bindWhere("relationId", id).bindWhere("type", type).search();
