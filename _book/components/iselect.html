移动执法3.0开发文档

iselect 高级下拉框组件

基本使用

<!--可以使用select-->
<select render="iselect">
    <option>--请选择--</option>
    <option value="1">广西壮族监察总队</option>
    <option value="2">乐业县监察支队</option>
    <option value="3">班德尔城提莫小分队</option>
</select>
<!--可以使用也可以用input-->
<input type="text" render="iselect" options="data:[{DM: '1', DMNR: '广西壮族监察总队'},{DM: '2', DMNR: '乐业县监察支队'},{DM: '3', DMNR: '班德尔城提莫小分队'}]">

切换单选和多选

<select render="iselect" multiple="multiple">
    <option>--请选择--</option>
    <option value="1">广西壮族监察总队</option>
    <option value="2">乐业县监察支队</option>
    <option value="3">班德尔城提莫小分队</option>
</select>

<input type="text" render="iselect" options="data:[{DM: '1', DMNR: '广西壮族监察总队'},{DM: '2', DMNR: '乐业县监察支队'},{DM: '3', DMNR: '班德尔城提莫小分队'}], multi:true">

从服务器端获取数据

<select render="iselect" multiple="multiple" options="textField:'DMNR',valueField:'DM',url:'${list}',params:{}">
    <option>--请选择--</option>
</select>

<input type="text" render="iselect" options="textField:'DMNR',valueField:'DM',url:'${list}',params:{} multi:true">

搜索功能开启

iselect 也支持搜索功能，会在下拉内容中加入一个搜索框, 当下拉数据过多时可以开启此功能

<select render="iselect" multiple="multiple" options="textField:'DMNR',valueField:'DM',url:'${list}',params:{},search:true">
    <option>--请选择--</option>
</select>

<input type="text" render="iselect" options="textField:'DMNR',valueField:'DM',url:'${list}',params:{} multi:true,search:true">

参数

  • url 远程加载数据的url地址
  • params 请求时参数
  • loadFilter 加载数据的过滤器，ajax请求成功后会调用这个方法，最终取这个方法 return 回来的数据。
  • data iselect的数据，如果不需要远程加载数据，写在这里即可
  • valueField 数据中值得字段默认是 DM
  • textField 数据中用于显示的文本字段， 默认是 DMNR
  • itemHeight 单个选项的高度，用户高度计算 默认 24
  • containerHeight 下拉框的最大高度 默认 240
  • search 是否开启搜索功能 默认 false
  • multi 是否开启多选功能 默认 false
  • onLoadSuccess 加载成功后回调 function(data){ } data是返回的数据
  • onSelect 选择某一项后回调 function(index, value, item){}

事件

select

用户选择某一项时触发事件

$(selector).on('iselect.select', function(e, index, value, item){

});
  • index 点击项对应数据的下标
  • value 点击项的value值
  • item 点击项数据整个object

loadSuccess

远程加载数据成功时触发事件

$(selector).on('iselect.loadSuccess', function(e, data){

});
  • data 服务器端返回的数据

方法

show

显示下拉内容

var iselect = $(selector).render('get');
iselect.show();

修改值和获取选择的值

$(selector).val('1');//单选
$(selector).val('1,2');//多选
$(selector).val();//获取值
