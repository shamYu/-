search 搜索框组件

搜索框，带有美化样式，记住历史搜索记录，和自动补全功能，在列表查询时常使用

基本使用

<input type="text" render="search"/>

记录搜索历史

上面的写法只能，出来搜索框的样式，想要加入记住搜索历史的功能还需要配置两个参数

  • name 保存在搜索历史的key
  • btnId 搜索按钮的id，会监听这个按钮的点击事件来存储数据
<input type="text" render="search" options="name:'searchText',  btnId:'searchBtn'"/>

自动补全搜索内容

搜索框还支持自动补全功能，需要配置 请求地址 和 参数 ,在输入时会发请求给后端，后端根据参数搜索，返回一些数据显示

<input type="text" render="search" options="name:'searchText',  btnId:'searchBtn' url:'${queryWryNameUrl}', filed:'WRYMC',params:{}"/>
  • url 请求地址
  • params 请求传递参数
  • field 返回object数组中取哪个字段值作为显示

服务器端配合如下:

request.getParameter("searchText"); //会传递这个参数，是搜索框的值
//服务器端接受参数
//省略查询逻辑
out.print("[{\"WRYMC\":\"test1\"},{\"WRYMC\":\"test2\"}]");

参数

name search存入cookie中的名称 btnId 搜索按钮id，配置之后自动启用保存搜索历史功能 url 请求地址，配置之后自动启用自动补全功能 params 请求时传递的参数 filed 返回object数组中取哪个字段值作为显示
