移动执法3.0开发文档

table相关的组件

render中定义三个table组件，会自动渲染定义的table(给 table 和里面的 td/th 自动加上样式)， 所有的 table 都不需要指定 options 参数

view-table

<table width="100%" render="view-table">
    <col width="20%"/>
    <col width="30%"/>
    <col width="20%"/>
    <col width="30%"/>
    <tr>
        <td></td>
        <td></td>
        <td></td>
        <td></td>
    </tr>
</table>

示例页面： webapp/pages/ydzfv3/yyyd/wrygk/glsx.jsp

query-table

query-table(查询表格)组件继承于view-table会自动给td和table添加样式，在这个基础上会自动给表单元素加上相应渲染器, 再列表页面中会用到，query-table中的表单会显得大一点，

<table width="100%" render="query-table">
    <col width="20%"/>
    <col width="30%"/>
    <col width="20%"/>
    <col width="30%"/>
    <tr>
        <td>
            <input type="text" placeholder="污染源名称/地址" id="search" name="searchText"  name="searchText" render="search" /><!--search是特殊组件无法自动渲染-->
        </td>
        <td></td>
        <td>   
                <select id="SSQX" name="SSQX">
                    <option value="">---请选择---</option>
                </select>
        </td>
        <td></td>
    </tr>
</table>

示例页面： webapp/pages/ydzfv3/xxwh/yyyd/wryjbxxList.jsp

form-table

form-table(表单) 同样继承与 view-table ，会给 table / td 标签加上样式，并且兼容我们的半自动表单

<table width="100%" render="form-table">
    <col width="20%"/>
    <col width="30%"/>
    <col width="20%"/>
    <col width="30%"/>
   <tr>
       <td>环保监管级别：</td>
       <td>
           <html:select property="HBJGJB">
               <c:forEach var="item" items="${applicationScope.SYS_CODE['HBJGJB']}">
                   <html:option value="${item.codeId}">${item.value}</html:option>
                       </c:forEach>
           </html:select>
       </td>
       <td>是否重点源：</td>
       <td>
           <html:select property="SFZDY">
                <html:option value="">--请选择--</html:option>
                <html:option value="1">是</html:option>
                <html:option value="0">否</html:option>
            </html:select>
        </td>
    </tr>
</table>

对半自动表单的支持， 因为，自定义的jsp标签遇到不存在的属性就会报错，所有我们有要渲染的组件就不能写 render 属性指定了，我们可以写在jsp标签的 styleClass 属性里，并加载 render- 前缀，至于渲染所需的参数写在 options 属性中即可

<html:select styleClass="render-iselect" options="multi:false">
    <html:option></html:option>
</html:select>

示例页面： webapp/pages/ydzfv3/xxwh/yyyd/wrygk/glsxEdit.jsp
