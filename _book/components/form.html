基本表单组件

在 table相关组件 中有提到 query-table 和 form-table 会自动扫描自己内部的表单元素，加上组件渲染标识(其实就是加上render属性),能自动天的的元素和对应的组件如下:

input type = text/password text组件 input type = radio radio组件 input type = checkbox checkbox组件 textarea textarea组件 select select组件

这些自动渲染的组件加上 date 、input-button unit-text 这些组件功能较少，基本只是封装了一些dom结构和样式, 或是有些简单的js, 这些组件如果不适用render直接写dom结构也是可以的. 但是有两点需要注意：

1.所有表单组件默认宽度为100%, 如果要手动调整，再元素上写 width 属性即可 2.select组件 如果是联动下拉框，改变了select里面的dom结构需要调用方法 trigger('change') 这样显示才会刷新

text 基本输入框美化

<!--用render的方式-->
<input type="text" name="name" render="text"> <!--如果在form-table/query-table下则不用指定render属性-->
<!--不用render的方式-->
<input type="text" name="name" class="form-input text">

radio 单选按钮美化

<!--必须用这种label+input的结构，会对这些做美化-->
<label>
    <input type="radio" render="radio" name="flag" value="1">
    <span>是</span>
</label>
<label>
    <input type="radio" render="radio" name="flag" value="0">
    <span>否</span>
</label>

checkbox 多选按钮美化

<label>
    <input type="checkbox" render="checkbox" name="type" value="01">
    <span>废水</span>
</label>
<label>
    <input type="checkbox" render="checkbox" name="type" value="02">
    <span>废气</span>
</label>
<label>
    <input type="checkbox" render="checkbox" name="type" value="03">
    <span>固废</span>
</label>
<label>
    <input type="checkbox" render="checkbox" name="type" value="04">
    <span>噪音</span>
</label>

textarea 文本域美化

textarea 本身拥有根据输入内容多少自动扩展高度的功能，不会是文本域里出现滚动条

<textarea render="textarea"></textarea>

select 下拉框美化

<select name="" render="select">
    <option value=""></option>
</select>

select 的美化改变这个dom结构，如果我们需要动态加载option时， 再加载完成之后一定要调用 $(select).trigger('change') 让组件重新渲染结构

date 时间输入

<input type="text" render="date" options="dateFmt:'yyyy-MM-dd HH:mm'"/>

date 组件是基于my97的封装，所有的配置项请参考 my97文档

input-button 输入框 + 按钮 显示样式

<div class="form-group">
    <input type="text" class="form-input text"/>
    <span class="input-group-btn">
        <button class="btn btn-select">...</button>
    </span>
</div>

unit-text 带单位显示输入框

<input type="text" render="unit-text" unit="(M)"/>
