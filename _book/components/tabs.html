移动执法3.0开发文档

tabs 选项卡组件

基本使用

<div render="tabs">
    <div class="box-panel" title="基本信息" name="jbxxPanel">
        this is pane1
    </div>
    <div class="box-panel" title="人员情况" name="ryqkPanel">
        this is pane2
    </div>
    <div class="box-panel" title="业务情况" name="ywqkPanel">
        this is pane3
    </div>
</div>

name 必须保证一个tabs组件下的没有相同 name 的panel

动态加载url

如果想让每一个panel都加载一个页面，这样需要配置一个url地址，它会自定加载页面， panel的参数都写在 panel-options 里

为什么是panel-options不是options？ 最开始api没有设计好，后来改版了要兼容以前的所以还是用了这个属性

<div render="tabs" id="tabs">
    <div class="box-panel" title="基本信息" name="jbxxPanel" panel-options="url:'${ctx}/pages/ydzfv3/xxwh/ydyd/Ydydjbxx.jsp'">
    </div>
    <div class="box-panel" title="人员情况" name="ryqkPanel" panel-options="url:'${ctx}/pages/ydzfv3/xxwh/ydyd/TxzzfRyxxlist.jsp'">
    </div>
    <div class="box-panel" title="业务情况" name="ywqkPanel" panel-options="url:'${ctx}/pages/ydzfv3/xxwh/ydyd/ydydywxx.jsp'">
    </div>
</div>

tabs 内置了 loadPage 的功能，也可以支持iframe加载只需设置 iframe:true 即可

固定tab菜单

如果每个panel的页面过长，再滚动的时候看不到这个顶部的选项卡，则可以固定顶部，让它不管如何滚动都在顶部

<div render="tabs" id="tabs" options="pin: 1">
    <div class="box-panel" title="基本信息" name="jbxxPanel" panel-options="url:'${ctx}/pages/ydzfv3/xxwh/ydyd/Ydydjbxx.jsp'">
    </div>
    <div class="box-panel" title="人员情况" name="ryqkPanel" panel-options="url:'${ctx}/pages/ydzfv3/xxwh/ydyd/TxzzfRyxxlist.jsp'">
    </div>
    <div class="box-panel" title="业务情况" name="ywqkPanel" panel-options="url:'${ctx}/pages/ydzfv3/xxwh/ydyd/ydydywxx.jsp'">
    </div>
</div>

只要配置了 pin 值就会自动启动这个功能

参数

tabs参数

  • removeMsg 删除panel时的提示信息 默认 确定删除？
  • scoller 需要固定顶部时监听滚动事件的dom 默认 window
  • pin 固定顶部时与窗口顶部的距离 默认 0

panel参数

  • show 是否默认显示 默认 false
  • name panel的name在单个tabs中唯一
  • title 显示标题
  • url 需要远程加载页面时的url路径
  • immediately 是否立即加载，如果设为true则在组价加载时就加载此页面（默认是在显示时才加载页面）默认 false
  • iframe 是否使用iframe加载 默认 false
  • canRemove 是否可以被删除 默认 false
  • onremove panel被删除时的回调函数
  • onloadSuccess 页面加载成功时的回调
  • onshow panel显示时的回调函数
  • btns: [] tabs组件最有边的按钮，当panel显示是，配置的按钮才会显示，[{text:'', icon:'', click: function()}] 这个参数已经过时，尽量不要使用

方法

获取tabs组件的一些状态

var tabs = $('#tabs').render('get');
tabs.showName //当前显示panel的name
tabs.panels   //所有的panel对象的集合
tabs.tabs     //获取当前所有tab的集合 tab = menu + btn + panel

show([name])

显示某个panel name是 panel 名称

var tabs = $('#tabs').render('get');
tabs.show()

addPanel([name,text, options, content])

添加一个panel到tabs组件中，

  • name panel名称
  • text 显示的title
  • options panel的参数
  • content 组件的dom内容，可以是一段html也可以是dom元素

removePanel([name])

删除指定 name 的panel

refreshPanel([name, url])

根据传入的url刷新指定 name 的panel，panel必须是远程加载的内容的，不然会报错
