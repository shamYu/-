移动执法3.0开发文档

datagrid 模块

所有操作 datagrid 的模块的基础模块， 封装了一些对于 datagrid 参数配置，所有 easyui datagrid 的参数都可以写在模块定义里, 所有的 on 开头的事件里， 在子模块中直接写 handler + 事件名, 这样就行。

基础参数

参数名 默认 描述
gridSelector '#datagrid' datagrid 选择器
tableInfo null tableInfo 配置
url null 加载datagrid的 url
idField 'XH' id字段
autoGridHeight false 是否自动延伸高度
lazyLoad false 是否延迟加载，设置为true之后需要手动调用 loadGrid 方法才会加载

剩下的都是 easyui datagrid的参数

事件处理

easyui datagrid 所有的事件这里都有封装， 调用的时候只需要吧 on改成 handler 定义在模块内即可

render.module('gridDemo', 'demo', {
    handlerLoadSuccess: function(){
        console.log('datagrid 加载成功后调用着这里');
    },
    handlerDblClickRow: function(){
        console.log('双击datagrid 上的行时调用这里');
    }
})

datagrid 方法

loadGrid

加载 datagrid，此方法在 lazyload 为 false 时，会在模块加载时自动调用。

resize

重新计算 datagrid 的宽高。

模块内的属性

this.grid

模块内 gridselector配置的 jquery 对象 会在模块 init 的时候生成。
