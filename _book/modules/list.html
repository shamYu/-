列表模块

列表模块封装了，查询，删除，显示详情页功能， 此模块继承 datagrid 模块需要配合easyui的datagrid 插件使用。另外它和 GenericService 中的 list 方法高度耦合

<nbean:ajaxUrl var="listUrl" clazz="com.szboanda.ydzf.wrygl.services.WryglService" method="list" parameters="sql=LIST_HJXF"/>
<nbean:tableInfo table="T_WRY_HJXF" pk="XH"/>
<div class="box-wrapper" render-module='list:{initParams:[{key : "WRYBH",value : "${WRYBH}"}],url:"${listUrl}",tableInfo:${tableInfo},detailUrl:"${ctx}/pages/ydzfv3/xxwh/yyyd/hjywxx/hjxfDetail.jsp?WRYBH=${WRYBH}"}'>
       <div id="listDiv">
           <div class="tr">
               <button class="btn list-add-btn"><span class="btn-icon icon-add">&nbsp;</span>新增</button>
               <button class="btn list-edit-btn"><span class="btn-icon icon-edit">&nbsp;</span>编辑</button>
               <button class="btn list-del-btn"><span class="btn-icon icon-remove">&nbsp;</span>删除</button>
           </div>
           <div class="list-box mt10">
               <table id="datagrid" width="100%">
                   <thead>
                   <tr>
                       <th data-options="field:'WRYBH',width:120,sortable:true,align:'up',halign:'center',hidden:true">污染源编号</th>
                       <th data-options="field:'XFBH',width:100,sortable:true,align:'center',halign:'center'">信访编号</th>
                       <th data-options="field:'BTSWRY',width:120,sortable:true,align:'center',halign:'center'">被投诉单位</th>
                       <th data-options="field:'TSRXM',width:120,sortable:true,align:'center',halign:'center'">投诉人姓名</th>
                       <!--省略一些-->
                   </tr>
                   </thead>
               </table>
           </div>
       </div>
       <div id="detailDiv">
       </div>
   </div>

基础参数

参数名 默认 描述
initParams null 初始化查询参数，sqlParam对象数组
autoGridHeight 10 默认开启自适应高度，距离底部的padding为10
searchSelector '#query-table' 查询容器的选择器
showDetailType 'loadPage' 显示详细页方式 loadPage用loadpage加载页面到本页面 dialog 对话框的方式打开 window 以window.open的方式打开
detailUrl null 详细页的url
appendParams ['XH'] 需要传递给详情页的参数名，会在调用详情页的时候拼在url后面（在grid field中的）
dialogTitle '弹出框' dialog标题
dialogWidth 800 dialog 宽度
dialogHeight 600 dialog 高度
addBackBtn true 是否自动添加返回按钮
backBtnContainerSelector '.tr:eq(0)' 返回按钮的容器
backBtnTpl '' 返回按钮的 html 模板
listDivSelector '#listDiv' listDiv 选择器
detailDivSelector '#detailDiv' detail的选择器

方法

search

调用的common.js中的 search 方法，查询数据

show([rowData])

显示详情页方式，rowData 是选择哪一行的数据，新增的时候不传递这个。

refresh([params])

重载 datagrid， params 是传递给后台的参数，sql params对象数组。

remove([rowData])

删除数据方法， 当没有配置 tableInfo 的时候会调用此方法，传入 rowdata，这时需要自行完成这个方法的逻辑

back()

由详情页返回列表页, 仅仅在 loadPage 加载详情时有用
