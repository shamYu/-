移动执法3.0开发文档

commonJs

这个js 封装了许多工具操作的库

datagrid操作

datagrid 自适应宽搞

$(datagrid).datagrid({
    onLoadSuccess: function(){
        common.grid.autoHeight(this, 15);
    }
})

common.grid.autoHeight(element, padding) 自适应宽高

  • element 元素或选择器
  • padding 距离底部的值， 默认是 10；

common.grid.autoResize(element, padding) 自适应宽度

$(datagrid).datagrid({
    onLoadSuccess: function(){
        common.grid.autoResize(this, 15);
    }
})

数据format

可以格式化，combobox， date, select, iselect 多种组件数据

<th data-options="field:'WRWDM', width:'80',sortable:true,align:'left',width:100,halign:'center',formatter:common.grid.format,editor:{type:'iselect',options:{valueField:'WRWDM',textField:'WRW',search:true,data:power.ajaxRequest('${queryWrwUrl}'),required:true}}"><span class="star">*</span>污染物名称</th>

form 操作

获取表单数据

common.form.getFormsData(formId)

参数 formId ， 默认是传入id， 也可以是 element或者jQuery 对象.

最后把元素的 name 和 value 为一个Object 返回

填充表单

common.form.fillForm(formId, data)

formId 默认 表单id， 也可以是 element或者jQuery 对象. data 填充的额数据

会扫描元素（表单和 span元素带 name属性） 的 name 跟 data 中的 key匹配， 最终将值填入元素中（.val() / .text() 方法写入）。

清空表单

common.form.clearForm(formId);

参数 formId ， 默认是传入id， 也可以是 element或者jQuery 对象.

验证表单

common.form.validate(formId);

参数 formId ， 默认是传入id， 也可以是 element或者jQuery 对象.

验证表单，返回验证结果 true/false;

tpl 模板替换


var tpl = '<span>这是条测试信息， 你看 名称: {name}, 年龄{age}</span>';
var data = {
    name: 'myron',
    age: 22
};
var arr = [data, {
    name: 'Lin',
    age: 20
}];

common.tpl.replace(tpl, data); //<span>这是条测试信息， 你看 名称: myron, 年龄22</span>
common.tpl.replace(tpl, arr);
//<span>这是条测试信息， 你看 名称: myron, 年龄22</span>
//<span>这是条测试信息， 你看 名称: Lin, 年龄20</span>

common.tpl.replace 会自动替换其中的属性值,如果嵌套过深 {a.b.c} 这样也是可以识别的

图片自适应

//common.ImgAutoSize(element, width, height, callback)

common.ImgAutoSize(element, 1000, 700, function(){
    //改变完成
})

此方法会把图片做等比例缩放，不会使图片失真，

element img元素 width 宽 height 高 callback 完成之后的回调函数

common.search

common.search(datagrid);

配合 datagrid 和 GenericService list 方法的查询方法， 组成 sql param 对象然后调用datagrid 的 reload 方法。
